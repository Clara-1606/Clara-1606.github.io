Proyecto Vue.js Wikispace


El objetivo de este ejercicio es aprender a utilizar el marco Vue.js 3.

Tuvimos que crear una wiki sobre astronautas y estaciones espaciales, desde una API externa ( https://spacelaunchnow.me/api/3.3.0/ ).

La aplicación se realiza en Vue.js 3 con la herramienta Quick. No hay un servidor Node.js ya que solo estoy usando una única llamada API externa.

El objetivo es mostrar la lista de astronautas con sus fotos, con un sistema de paginación.

Tenemos 3 pestañas, por defecto, estamos en la lista de astronautas.

  • La lista de astronautas
  • La lista de embarcaciones
  • La lista de eventos

Una vez que se clona el proyecto, debe instalar las dependencias. Nos metemos en la carpeta y hacemos: npm i. Luego lanzamos el proyecto: npm run dev
