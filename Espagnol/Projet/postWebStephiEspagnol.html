Proyecto sitio de agencia inmobiliaria


Cada final de curso en Ynov tenemos un gran proyecto (UF) de varios meses para llevarlo a cabo en equipo para volver al final del curso para poner en práctica todo lo aprendido a lo largo de nuestro formación.

Este año el tema fue ayudar a la empresa StephiPlace, un grupo de agencias inmobiliarias, a expandirse en Francia.

Para esos 3 proyectos distintos:

  • Un sitio web con anuncios
  • Un software para agentes inmobiliarios
  • Infraestructura de red para agencias

El primer proyecto fue el sitio web donde se puede ver la página de inicio.

Este sitio permite a un cliente (comprador / vendedor o ambos) conectarse, tener su cuenta con su información personal que puede modificar, ver sus anuncios, ver sus favoritos. Puede colocar un anuncio de una propiedad o realizar una búsqueda con diferentes criterios.

Este sitio también fue creado desde cero, es decir completamente hecho a mano sin framework, ni plantillas, en HTML / CSS, JavaScript, SQL y PHP. Está vinculado a una base de datos MySQL.
