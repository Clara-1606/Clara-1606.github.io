Proyecto Laravel Todo


Para aprender a usar Laravel 8, teníamos un proyecto en el que teníamos que hacer una aplicación web de gestión de tareas colaborativa como Trello.

Para utilizar esta aplicación, el usuario debe estar registrado e iniciado sesión. Una vez que haya iniciado sesión, puede ver / crear / modificar / eliminar tareas y agregar otros usuarios, que también están registrados.

Una tarea tiene título, descripción, fecha de finalización, propietario (solo él puede eliminarla), prioridad, estado, categoría y puede tener varios documentos adjuntos.

Se puede asignar una tarea a uno de estos participantes.

Cada usuario puede agregar comentarios sobre las tareas. Se debe mantener su nombre y la hora y fecha del comentario, y es posible editar el comentario (esto se marcará en el comentario que ha sido editado) e incluso borrarlo.

Para cada parte, están todas las acciones de CRUD.

También hemos comenzado a gestionar las autorizaciones con las políticas para no tener acceso al tablero que no nos pertenece.

El objetivo principal era realmente trabajar en la parte posterior de la aplicación, el frente no funcionaba mucho, usé Bootstrap para una representación simple y limpia rápidamente.
