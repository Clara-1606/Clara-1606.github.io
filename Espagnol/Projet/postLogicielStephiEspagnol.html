Proyecto de software para agentes inmobiliarios


Cada final de curso en Ynov tenemos un gran proyecto (UF) de varios meses para llevarlo a cabo en equipo para volver al final del curso para poner en práctica todo lo aprendido a lo largo de nuestro formación.

Este año el tema fue ayudar a la empresa StephiPlace, un grupo de agencias inmobiliarias, a expandirse en Francia.

Para esos 3 proyectos distintos:

  • Un sitio web con anuncios
  • Un software para agentes inmobiliarios
  • Infraestructura de red para agencias

El segundo proyecto fue el software en Python con Tkinter para la interfaz gráfica. Estaba acostumbrado a esta tecnología habiendo realizado ya un gran proyecto de fin de año en ISN (Ciencias de la Computación y Ciencias Digitales) durante mi último año de secundaria. Había podido hacer un Mastermind en Python también con Tkinter.

Era mucho más complicado porque tenía que vincular el proyecto a la base de datos y trabajar con él.

El propósito de este software era que los agentes inmobiliarios pudieran publicar anuncios, ver los anuncios que ya habían publicado o no, y poder modificarlos / eliminarlos. También podría crear una nueva cuenta para otro agente o modificar las tarifas de la agencia.
