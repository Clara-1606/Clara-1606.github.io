Projet Vue.js Wikispace


Le but de cet exercice est d’apprendre à utiliser le framework Vue.js 3.

Nous devions réaliser un wiki sur les astronautes et les stations spatiales, à partir d’une API externe ( https://spacelaunchnow.me/api/3.3.0/ ).

L’application est réalisée en Vue.js 3 avec l’outil Vite. Il n’y a pas de serveur Node.js étant donné que j’utilise uniquement un appel à une seule API externe.

Le but est d’afficher la liste des astronautes avec leurs photos, avec un système de pagination.

Nous avons 3 onglets, par défaut, on est sur la liste des astronautes.

  • La liste des astronautes
  • La liste des vaisseaux
  • La liste des évènements

Une fois le projet cloné, il faut installer les dépendances. On se met dans le dossier et on fait : npm i . Ensuie on lance le projet : npm run dev
