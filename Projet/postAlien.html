Projet Space Invaders


En fin d’année nous devions améliorer un code en Python qui n’était pas le notre pour ajouter des fonctionnalités.

Nous avions le code d’un jeu célèbre le Space Invaders avec le minimum où nous pouvions déjà jouer mais où nous devions par exemple rajouter un système de points, rajouter le vaisseau spécial qui vient quand on a gagné une partie, faire tirer les aliens sur notre vaisseau pour avoir une autre façon de perdre le jeu.

Ce fût une expérience particulière car il fallait d’abord comprendre comment fonctionnait le code pour pouvoir ainsi le modifier et l’améliorer.

C’est à ce moment où on voit l’importance des commentaires.
